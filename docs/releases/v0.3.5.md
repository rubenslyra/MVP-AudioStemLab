# Release v0.3.5 - CPU and CUDA Portable Packages

This release separates the desktop packages into CPU and CUDA variants.

## Why This Matters

CUDA is not required for every computer. It only helps on Linux/Windows machines with a compatible NVIDIA GPU and driver stack. On machines without NVIDIA CUDA support, CUDA libraries make the package much larger without improving performance.

## Recommended Download

Use the CPU package unless you know the machine has a compatible NVIDIA GPU:

```text
AudioStemLab-v0.3.5-linux-cpu.zip
AudioStemLab-v0.3.5-windows-cpu.zip
AudioStemLab-v0.3.5-macos-cpu.zip
```

Use the CUDA package only for supported NVIDIA systems:

```text
AudioStemLab-v0.3.5-linux-cuda.zip
AudioStemLab-v0.3.5-windows-cuda.zip
```

macOS is CPU-only.

## End User Requirements

End users do not need to install Python, create a virtual environment, or run `pip`.

Download the ZIP for the operating system, extract it, and open the matching launcher.

## Notes

Demucs model weights may still be downloaded on first use if they are not already available in the user's model cache.
