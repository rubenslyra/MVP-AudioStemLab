# Release v0.3.4 - Experiencia de Terminal para Testadores

Esta versao melhora a primeira experiencia de uso da versao de teste desktop do RLLABS Audio Stem Lab.

## Plataformas

Disponivel apenas para desktop:

- Linux
- Windows
- macOS

Mobile ainda nao esta disponivel.

## Destaques

- Pacotes portaveis por sistema operacional via GitHub Releases.
- Usuarios finais nao precisam instalar Python, criar virtualenv ou rodar `pip`.
- Terminal com visual mais profissional usando `rich`.
- Cabecalho persistente com identidade do sistema, versao e plataforma.
- Limpeza de interacoes entre telas sem perder informacoes cruciais.
- Seletor nativo para escolher arquivo de origem.
- Seletor nativo para escolher pasta de destino.
- Mensagens de progresso com percentual inteiro e tempo decorrido.
- Inclusao da fonte Fira Code v6.2 no pacote.
- Launchers auxiliares para Linux, Windows e macOS.

## Observacoes Sobre Fonte e Ligaduras

A fonte Fira Code esta incluida no pacote em:

```text
assets/fonts/fira-code/ttf/
```

A aplicacao prepara o asset, mas a ativacao de fonte e ligaduras depende do terminal ou do instalador de cada sistema operacional.

## Como Rodar

Baixe o ZIP correspondente ao seu sistema operacional na pagina de release, extraia o pacote e use um dos launchers:

```text
launchers/audiostemlab-linux.sh
launchers/audiostemlab-linux.desktop
launchers/audiostemlab-macos.command
launchers/audiostemlab-windows.bat
```

O pacote inclui o runtime da aplicacao. Nao instale Python para usar a versao de teste.

Observacao: os pesos dos modelos Demucs ainda podem ser baixados no primeiro uso caso nao estejam no cache do usuario.
