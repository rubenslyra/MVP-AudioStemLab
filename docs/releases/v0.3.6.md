# Release v0.3.6 - Portable Download Artifacts

This release publishes ready-to-download desktop packages through the repository release page.

## Recommended Download

Use the CPU package unless you know the machine has a compatible NVIDIA GPU:

```text
AudioStemLab-v0.3.6-linux-cpu.zip
AudioStemLab-v0.3.6-windows-cpu.zip
AudioStemLab-v0.3.6-macos-cpu.zip
```

Use the CUDA package only for supported NVIDIA systems:

```text
AudioStemLab-v0.3.6-linux-cuda.zip
AudioStemLab-v0.3.6-windows-cuda.zip
```

macOS is CPU-only.

## End User Requirements

End users do not need to install Python, create a virtual environment, or run `pip`.

Download the ZIP for the operating system, extract it, and open the matching launcher.

## Notes

Demucs model weights may still be downloaded on first use if they are not already available in the user's model cache.
