# Release v0.3.8 - Portable Package Experience Fixes

This release fixes first-run problems found while testing the Linux CPU package from `v0.3.7`.

## Recommended Download

Use the CPU package unless you know the machine has a compatible NVIDIA GPU:

```text
AudioStemLab-v0.3.8-linux-cpu.zip
AudioStemLab-v0.3.8-windows-cpu.zip
AudioStemLab-v0.3.8-macos-cpu.zip
```

Use CUDA only for supported NVIDIA systems:

```text
AudioStemLab-v0.3.8-linux-cuda.zip.part-*
AudioStemLab-v0.3.8-windows-cuda.zip.part-*
```

macOS is CPU-only.

## What Changed

- Portable launchers now open the bundled PyInstaller executable inside the ZIP package.
- The Linux `.desktop` launcher now points to the package launcher correctly and is executable.
- `yt-dlp` is collected into the PyInstaller build and called through the packaged Python runtime.
- The home screen no longer shows internal app folders before the user chooses a file or folder.
- Download flow asks where to save the authorized audio before starting the download.
- Screen clearing now also clears terminal scrollback when the terminal supports it.
- Exit no longer leaves the countdown/finalization text in the terminal.

## CUDA Package Notes

GitHub release assets cannot exceed 2 GiB. CUDA packages are larger than that, so they are published as multiple `*.part-*` files.

Download every part for the CUDA package you need, keep them in the same folder, and rebuild the ZIP before extracting it.

Linux or macOS:

```bash
cat AudioStemLab-v0.3.8-linux-cuda.zip.part-* > AudioStemLab-v0.3.8-linux-cuda.zip
```

Windows PowerShell:

```powershell
cmd /c copy /b AudioStemLab-v0.3.8-windows-cuda.zip.part-aa+AudioStemLab-v0.3.8-windows-cuda.zip.part-ab AudioStemLab-v0.3.8-windows-cuda.zip
```

If more Windows CUDA parts are present, include each part in alphabetical order in the `copy /b` command.

## End User Requirements

End users do not need to install Python, create a virtual environment, or run `pip`.

Download the package for the operating system, extract it, and open the matching launcher.

## Notes

Demucs model weights may still be downloaded on first use if they are not already available in the user's model cache.
