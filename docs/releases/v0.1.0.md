# Release v0.1.0 - MVP CLI

Esta é a primeira versão versionada do RLLABS Audio Stem Lab.

## Para Usuários

Esta versão ainda é voltada a execução local via Python, com base pronta para empacotamento futuro.

### Instalação

Use o ambiente isolado já preparado no projeto:

```bash
source .venv311/bin/activate
python app.py
```

Em uma instalação nova de desenvolvimento:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip wheel
pip install -r requirements.txt
python app.py
```

Use preferencialmente Python 3.10 ou 3.11.

## Funcionalidades

- Separação local de áudio com Demucs.
- Modos de separação:
  - voz + instrumental;
  - voz + bateria + baixo + outros;
  - voz + bateria + baixo + guitarra + piano/teclas + outros.
- Formatos de saída:
  - WAV;
  - MP3 320 kbps;
  - FLAC.
- Download opcional com `yt-dlp` para áudios próprios, autorizados, Creative Commons ou domínio público.
- Saída em `output_stems/` durante desenvolvimento.

## Limitações Conhecidas

- Ainda não há instaladores finais para Linux, Windows e macOS.
- O build PyInstaller está scaffoldado, mas ainda precisa ser validado por plataforma.
- O ambiente atual instalou PyTorch com CUDA; a próxima versão deve priorizar uma instalação CPU-only mais leve.
- Sopros, sintetizadores e backing vocals ainda não são separados como stems próprios confiáveis.
